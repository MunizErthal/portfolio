Resumo

Meu site de contos foi desenvolvido em 2021 com a ideia de um local publico para compartilhar minhas histórias, você pode acessar ele aqui: fernandome.com

A ideia era criar um local focado na leitura de histórias, com features que ajudassem os leitores a terem uma experiencia agradável durante a leitura

No mercado não existiam locais para publicação que oferecesse a liberdade que eu desejava, nem mesmo o controle que eu queria com minhas publicações, então por isso resolvi desenvolver meu próprio site

Dessa forma surgiu o meu site de contos, onde todas as publicações são histórias minhas, com features que agregam como um sistema de leitura e compartilhamento de histórias.

Com um layout influenciado pela Netflix o desejo era criar uma espécie de biblioteca de contos para eu poder promover minhas histórias

Funcionamento

Os usuários podem acessar o site e lerem qualquer história publicada.

É possível curtir e comentar as histórias

Além disso, os usuários podem compartilhar trechos das histórias, com capas personalizadas em suas redes sociais

Existem várias ferramentas para facilitar a leitura, como o modo escuro, configuração do tamanho da fonte e barra de progresso da leitura.

Tecnologias e Implementação

O projeto foi desenvolvido com Angular 9 e TypeScript, utilizando HTML e CSS para a construção de um front-end responsivo.

Para armazenar os contos, comentários e curtidas, foi escolhido o Firebase como banco de dados, uma vez que a ideia não era implementar um serviço de back-end, já que a solução final seria simples

O projeto foi hospedado no Google Cloud Platform.

Além da leitura de contos, outras funcionalidades como pesquisas por tags, títulos e genêros, ou o compartilhamento de trechos dos contos, foram totalmente implementados no front-end.

Recursos

  • Leitura dos contos publicados
  • Layout responsivo para leitura em smartphones
  • Pesquisa por título, gênero e tags
  • Informativo de número de leitura/curtidas por publicação
  • Modos de leitura Dark Mode e Light Mode
  • Barra de progresso da leitura
  • Controle do tamanho da fonte na leitura
  • Seleção de trechos para compartilhamento
  • Compartilhamento em diferentes formatos e com opções de artes
  • História com trechos com opção de tradução
  • Botão para curtir histórias
  • Inclusão de comentários

Layout responsivo para leitura em smartphones

Um dos focos aqui era criar um layout responsivo para uma leitura agradável a partir de dispositivos móveis.

Pesquisa por título, gênero e tags

Um dos recursos que eu desejava era a facilidade em encontrar as histórias, ou navegar de acordo com genêros ou tags relacionadas.

Modos de leitura Dark Mode e Light Mode e Barra de progresso da leitura

O modo de leitura também era algo muito importante pra mim, uma vez que vários dos sites de publicação que eu encontrei não tinham nem mesmo este recurso.

Controle do tamanho da fonte na leitura

Controlar o tamanho da fonte para facilitar a leitura de usuários que poderiam ter problema com o tamanho da fonte.

Seleção de trechos para compartilhamento e compartilhamento em diferentes formatos e com opções de artes

A ideia aqui era facilitar o compartilhamento das histórias, para que os usuários pudessem dar apoio na divulgação do trabalho.

História com trechos com opção de tradução

Incluir trechos de outros idiomas, permitindo uma leitura mais imersiva, mas, com a opção de tradução para o usuário que quer entender o que está sendo dito, seja em uma primeira leitura ou para leituras posteriores.
